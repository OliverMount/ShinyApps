## How to Run Locally

1. Install R and RStudio.  
2. Install required packages:

```r
install.packages(c("shiny", "ggplot2", "dplyr"))
```

3. Clone the repository

```
git clone https://github.com/OliverMount/ShinyApps.git

```

4. Run the app

```
library(shiny)
runApp("path/to/app_folder")
```
